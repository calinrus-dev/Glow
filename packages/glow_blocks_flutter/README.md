# glow_blocks_flutter

Widgets y bindings Flutter para el motor de bloques de Glow.

## Características
- Widgets para bloques
- Integración con Riverpod y Flutter
- Temas y estilos personalizables

## Uso
```dart
import 'package:glow_blocks_flutter/glow_blocks_flutter.dart';
```

## Arquitectura
Depende de Flutter y glow_blocks_engine.

## Tests
```sh
dart test packages/glow_blocks_flutter
```
