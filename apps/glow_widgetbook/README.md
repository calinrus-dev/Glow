# Glow Widgetbook

Simple playground to preview and test `glow_ui` components.

## Run

```bash
cd apps/glow_widgetbook
flutter pub get
flutter run -d windows  # or chrome, macos, etc
```

## Included Components

- **Buttons**: GlowButton (Primary, Secondary, Ghost), GlowPrimaryButton
- **Inputs**: GlowInput (label, hint, icons, error, helper)
- **Avatars**: GlowAvatar (sizes and border glow)
- **Colors**: Full Glow UI palette
