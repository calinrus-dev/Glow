# Contribuir a Glow

¡Gracias por tu interés en contribuir! Por favor, sigue estas pautas:

## Requisitos
- Usa ramas feature/ para nuevas funcionalidades
- Haz pull requests a main
- Escribe tests para nuevas features
- Sigue la guía de estilo de Dart/Flutter

## Flujo básico
1. Haz un fork y clona el repo
2. Crea una rama: `git checkout -b feature/mi-feature`
3. Haz tus cambios y tests
4. Haz commit y push: `git commit -m "feat: mi feature"`
5. Abre un Pull Request

## Código de conducta
Sé respetuoso y profesional. No se tolera contenido ofensivo.
